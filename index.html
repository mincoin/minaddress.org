A Mincoin wallet is as simple as a single pairing of a Mincoin address with it's corresponding Mincoin private key.

To safeguard this wallet you must print or otherwise record the Mincoin address and private key. It is important to make a backup copy of the private key and store it in a safe location.

This site does not have knowledge of your private key. If you are familiar with PGP you can download this all-in-one HTML page and check that you have an authentic version from the author of this site by matching the SHA1 hash of this HTML with the SHA1 hash available in the signed version history document linked on the footer of this site.

If you leave/refresh the site or press the Generate New Address button then a new private key will be generated and the previously displayed private key will not be retrievable.

Your Mincoin private key should be kept a secret. Whomever you share the private key with has access to spend all the Mincoins associated with that address.
